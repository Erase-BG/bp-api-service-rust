# BP API Service

The microservice for handling background removal request at [https://api.erasebg.org](https://api.erasebg.org).

## Environment variables

```markdown
MEDIA_ROOT=
MEDIA_URL=
MEDIA_SERVE_HOST=
BP_SERVER_HOST=
BP_SERVER_AUTH_TOKEN=
PROCESS_HARD=
POSTGRES_URL=
```

### Run

```shell
cargo run --release
```
